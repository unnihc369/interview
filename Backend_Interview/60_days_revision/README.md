## File naming convention

```
day_N/
  dayN_leetcode.md      — 3 DSA problems (Mon–Fri)
  dayN_java.md          — Java topic(s)
  dayN_spring_boot.md   — Spring Boot topic(s)
  dayN_lld.md           — LLD / design patterns
  dayN_hld_mysql.md     — Sat/Sun HLD + MySQL
```

Each file includes: problem statements, approaches, Java (+ JS for DSA), complexity, deep explanations, and interview Q&A.
